|icon| Neuroptikon is a tool for visualizing and interacting with neural circuits.

Circuits are defined using a biological model independent of any particular visualization.  Circuits can then be visualized in whatever way helps to answer the question at hand.  Multiple visualizations of the same network can even be used simultaneously.
