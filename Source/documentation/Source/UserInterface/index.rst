User Interface
==============

The primary interface of Neuroptikon is the :doc:`network window <NetworkWindow>`.  There is an :doc:`inspector <Inspector>` for viewing and modifying properties of objects and a window for :doc:`finding <Finder>` objects.  Lastly there is a :doc:`library <Library>` window which stores common items shared by all network windows.
